# Employee Attrition Prediction System

## 📌 Project Overview
This project predicts employee attrition using machine learning techniques based on HR analytics data. It helps identify employees who are likely to leave the company.

---

## 📊 Objectives
- Perform Exploratory Data Analysis (EDA)
- Clean and preprocess HR dataset
- Train multiple machine learning models
- Evaluate model performance
- Identify key factors affecting employee attrition

---

## 🧠 Machine Learning Models Used
- Logistic Regression
- Decision Tree
- Random Forest

---

## 📈 Key Steps
- Data Cleaning
- Feature Engineering
- Encoding categorical variables
- Model training and evaluation
- Visualization of insights

---

## 📊 Visualizations
Includes:
- Target distribution
- Correlation heatmap
- Feature importance
- Model comparison
- ROC curves

(Images are available in `attrition_plots/` folder)
